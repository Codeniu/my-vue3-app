# My Vue 3 App

## 项目介绍
这是一个使用 Vue 3 + TypeScript + Vite 构建的现代化前端项目。

## 技术栈
- Vue 3 
- TypeScript
- Vite
- Vue Router
- Pinia
- Axios

## 开发环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0

## 项目设置
1. 音频播放器组件，支持播放列表、进度控制、播放速率调整等功能
2. AI聊天机器人组件，支持新建对话、消息发送、流式响应、消息存储等功能
3. 项目配置文件，包括TypeScript、ESLint、Prettier等工具的配置
4. 基础页面布局和路由配置，包含Home和About两个视图
